Fuck Github, hello Codeberg

12rd of August, 2025

Been a long time since I’ve written a post on the blog, aye? Well, there’s a reason for that: I am way too lazy. I’ve started writing some posts about different themes but haven’t finished any single one.

I know this blog has accidentally been about a web journal with future plans that haven’t been finished. But hear me out on this one.

I’m remaking the website for the third time, now it will hopefully be flexible enough to allow for me to use different templates depending if I want the sidebar on one side, the other one or no sidebar at all; all while allowing me to display custom per-page content so I can link to a different language version of the page.

But you ghost reader may be wondering, why that title? Heh, due to growing concerns about Github’s leadership, I’ve decided to move to codeberg. Here’s a simple table so you quickly catch up.

Github Codeberg
Shoves copilot AI Has no AI
Has some trackers Does not track you
It’s proprietary software itself It’s free software itself
Owned by Microsoft Not-for-profit business

My main concerns are specially the AI stuff, microsoft’s ownership of the platform and the fact that it’s proprietary itself.

As far as I know, Github is still a good platform, hell, you are reading this because it’s hosted for now on Github pages. But not for long.

At the end of the day, you may use what it feels right, there’s no denying that, and probably I’m going schitzo with paranoia, but I’d rather use a service that is free software in itself and not to use any microsoft stuff.

Since I’m going to remake the website (for the third time lmao) I decided to publish it on codeberg instead since the pages services is all I need. Sure, the repository’s size limit may be 750Mb, but my website weights around 15Mb at the time of writing this. But come on, text takes next to no space, the heaviest things will be media files and turns out that .webp and .webm files are lighter than .png or .mp4 files. So in my case is totally fine.

This was just a quick post I decided to write on the spot. I literally discovered codeberg scrolling trough my Mastodon feed (follow me in there btw) and just made my account.

I managed to set up a custom domain for it and for now it works, if you want to check the progress you can check in here, but for now there’s nothing. I may change the domain to juangames.net, but I’m still not sure about it. Probably once I’m done rebuilding the site from scratch.
